Integrations
API Key: Change YOUR-API-KEY with your Linkup API key. Get your free API key
Cursor

Add to ~/.cursor/mcp.json:

{ "mcpServers": { "linkup-company-research": { "url": "https://company-research.linkup.tools/sse?apiKey="YOUR-API-KEY"" } } }
Claude Code
claude mcp add --transport http linkup-company-research https://company-research.linkup.tools/sse?apiKey="YOUR-API-KEY"
Claude Code (Local)

Run the server locally using uvx:

claude mcp add linkup-company-research -- uvx linkup-company-research-mcp

Then set your API key as an environment variable:

export LINKUP_API_KEY=YOUR-API-KEY
